# Tutorials

Hands-on guides that walk you through complete workflows.

## Section Contents

| Tutorial | Description | Time |
|----------|-------------|------|
| [Your First Docs Site](first-docs-site.md) | Create a documentation site from scratch | 15 min |

## Tutorial Format

Each tutorial includes:

1. **Goal** - What you'll build
2. **Prerequisites** - What you need before starting
3. **Steps** - Detailed instructions with code
4. **Verification** - How to test your work
5. **Next Steps** - Where to go from here

## Recommended Order

If you're new to mdbook-htmx, follow this learning path:

```
1. Your First Docs Site
         │
         ▼
2. Adding Search (optional)
         │
         ▼
3. Adding Authentication (if needed)
         │
         ▼
4. Production Deployment
```

## Prerequisites

Before starting tutorials, ensure you have:

- [Rust installed](https://www.rust-lang.org/tools/install)
- [mdbook-htmx installed](../getting-started/installation.md)
- A text editor
- Command line access
